Seaborn¶

Seaborn es una increíble biblioteca de visualización para el trazado de gráficos en Python. Proporciona hermosos estilos por defecto y paletas de colores para hacer que los gráficos sean más atractivos. Está construida sobre la biblioteca matplotlib y también está estrechamente integrada con las estructuras de datos de pandas.

Seaborn tiene como objetivo hacer que la visualización sea la parte central de la exploración y comprensión de los datos. Proporciona APIs orientadas al conjunto de datos, de modo que podemos cambiar entre diferentes representaciones visuales para las mismas variables para una mejor comprensión del conjunto de datos.

Algunos ejemplos desde la galería:¶

Diferentes categorías de gráficos en Seaborn¶

Los gráficos se utilizan básicamente para visualizar la relación entre variables. Esas variables pueden ser completamente numéricas o una categoría como un grupo, clase o división. Seaborn divide los gráficos en las siguientes categorías:

  • Gráficos relacionales: se utiliza para entender la relación entre dos variables.
  • Gráficos categóricos: se ocupa para variables categóricas y de cómo se pueden visualizar.
  • Gráficos de distribución: se utiliza para examinar las distribuciones univariantes y bivariantes.
  • Gráficos de regresión: pensados principalmente para añadir una guía visual que ayude a enfatizar los patrones en un conjunto de datos durante los análisis exploratorios de datos.
  • Gráficos de matriz: se utilizan para visualizar la dispersión de los datos.
  • Cuadrículas multigráficos: útil para dibujar múltiples instancias del mismo gráfico en diferentes subconjuntos del conjunto de datos.

Uso de Seaborn con Matplotlib¶

El uso conjunto de Matplotlib y Seaborn es un proceso muy sencillo. Sólo tenemos que invocar la función de trazado de Seaborn de forma normal, y luego podemos utilizar la función de personalización de Matplotlib.

In [ ]:
# importing packages
import seaborn as sns
import matplotlib.pyplot as plt

# loading dataset
data = sns.load_dataset("iris")

# draw lineplot
sns.lineplot(x="sepal_length", y="sepal_width", data=data)

# setting the title using Matplotlib
plt.title('Title using Matplotlib Function')

# setting the x limit of the plot
plt.xlim(5,7)

plt.show()

Personalización de los gráficos con Seaborn¶

Seaborn viene con algunos temas personalizados y una interfaz de alto nivel para personalizar el aspecto de los gráficos. Considere el ejemplo anterior donde se utiliza el valor por defecto del Seaborn. Todavía se ve bien y bonito, pero podemos personalizar el gráfico de acuerdo a nuestras propias necesidades. Así que vamos a ver el estilo de los gráficos en detalle.

  • Cambiando el estilo: set_style() se utiliza para establecer el estilo del gráfico. Esto significa que afecta a cosas como el color de los ejes, si la cuadrícula está activa o no, u otros elementos estéticos. Hay cinco temas disponibles en Seaborn: darkgrid, whitegrid, dark, white, ticks.

  • Cambiar el tamaño de la figura: El tamaño de la figura se puede cambiar usando el método figure() de Matplotlib. El método crea una nueva figura del tamaño especificado que se pasa en el parámetro figsize.

In [ ]:
# importing packages
import seaborn as sns
import matplotlib.pyplot as plt

# loading dataset
data = sns.load_dataset("iris")

# changing the figure size
plt.figure(figsize = (10, 4))

# Selecting style as white,
# dark, whitegrid, darkgrid
# or ticks
sns.set(style="white")

# draw lineplot
sns.lineplot(x="sepal_length", y="sepal_width", data=data)

# Removing the spines
sns.despine()

plt.show()
  • Escalado de los gráficos: Se puede hacer mediante el método set_context(). Anula los parámetros por defecto, afectando el tamaño de las etiquetas, líneas y otros elementos del gráfico, pero no al estilo general. El contexto base es "notebook", y los otros contextos son "paper", "talk", y "poster". font_scale establece el tamaño de la fuente.

  • El mapa de calor se define como una representación gráfica de datos que utiliza colores para visualizar el valor de la matriz, se puede trazar utilizando la función heatmap(). En ella, para representar los valores más comunes o las actividades más altas se utilizan colores más brillantes, básicamente rojizos, y para representar los valores menos comunes o de actividad se prefieren los colores más oscuros.

Plotly¶

Plotly ofrece los gráficos estándar que todos los usuarios de Python querrían utilizar, de manera visualmente estimulantes o interactivos. Plotly está haciendo grandes progresos en estas dos áreas. Además dan acceso a gráficos avanzados, centrados en ML y AI.

La verdadera innovación es el enfoque de Plotly en los gráficos interactivos. En lugar de tener que preocuparse de ajustar el código cada vez que quiera cambiar la vista, Plotly le ofrece una interfaz gráfica para hacerlo.

Lux¶

Lux es una gran herramienta para la visualización y el análisis exploratorio de datos. Es posible tomar cualquier marco de datos y recomendará automáticamente visualizaciones que le ayuden a explorar y compartir los datos. No es necesario que escriba usted mismo ningún código de visualización.

import lux
import pandas as pd

A continuación, se puede ver cómo Lux proporciona opciones de visualización para cualquier marco de datos. Todo lo que tienes que hacer es seleccionar el gráfico que quieres, sin necesidad de codificar nada. Esto es un gran ahorro de tiempo, ya que conseguir la sintaxis correcta exacta de un paquete de como Matplotlib o Seaborn puede ser intensivo en tiempo.
