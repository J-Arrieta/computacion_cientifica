Seaborn¶

Seaborn es una increble biblioteca de visualizacin para el trazado de grficos en Python. Proporciona hermosos estilos por defecto y paletas de colores para hacer que los grficos sean ms atractivos. Est construida sobre la biblioteca matplotlib y tambin est estrechamente integrada con las estructuras de datos de pandas.

Seaborn tiene como objetivo hacer que la visualizacin sea la parte central de la exploracin y comprensin de los datos. Proporciona APIs orientadas al conjunto de datos, de modo que podemos cambiar entre diferentes representaciones visuales para las mismas variables para una mejor comprensin del conjunto de datos.

Diferentes categorías de gráficos en Seaborn¶

Los grficos se utilizan bsicamente para visualizar la relacin entre variables. Esas variables pueden ser completamente numricas o una categora como un grupo, clase o divisin. Seaborn divide los grficos en las siguientes categoras:

  • Grficos relacionales: se utiliza para entender la relacin entre dos variables.
  • Grficos categricos: se ocupa para variables categricas y de cmo se pueden visualizar.
  • Grficos de distribucin: se utiliza para examinar las distribuciones univariantes y bivariantes.
  • Grficos de regresin: pensados principalmente para aadir una gua visual que ayude a enfatizar los patrones en un conjunto de datos durante los anlisis exploratorios de datos.
  • Grficos de matriz: se utilizan para visualizar la dispersin de los datos.
  • Cuadrculas multigrficos: til para dibujar mltiples instancias del mismo grfico en diferentes subconjuntos del conjunto de datos.

Uso de Seaborn con Matplotlib¶

El uso conjunto de Matplotlib y Seaborn es un proceso muy sencillo. Slo tenemos que invocar la funcin de trazado de Seaborn de forma normal, y luego podemos utilizar la funcin de personalizacin de Matplotlib.

import matplotlib.pyplot as plt

# loading dataset

# draw lineplot

# setting the title using Matplotlib

# setting the x limit of the plot
plt.xlim(5,7)

plt.show()

Personalización de los gráficos con Seaborn¶

Seaborn viene con algunos temas personalizados y una interfaz de alto nivel para personalizar el aspecto de los grficos. Considere el ejemplo anterior donde se utiliza el valor por defecto del Seaborn. Todava se ve bien y bonito, pero podemos personalizar el grfico de acuerdo a nuestras propias necesidades. As que vamos a ver el estilo de los grficos en detalle.

  • Cambiando el estilo: set_style() se utiliza para establecer el estilo del grfico. Esto significa que afecta a cosas como el color de los ejes, si la cuadrcula est activa o no, u otros elementos estticos. Hay cinco temas disponibles en Seaborn: darkgrid, whitegrid, dark, white, ticks.

  • Cambiar el tamao de la figura: El tamao de la figura se puede cambiar usando el mtodo figure() de Matplotlib. El mtodo crea una nueva figura del tamao especificado que se pasa en el parmetro figsize.

In [ ]:
# importing packages
import seaborn as sns
import matplotlib.pyplot as plt

# loading dataset
data = sns.load_dataset("iris")

# changing the figure size
plt.figure(figsize = (10, 4))

# Selecting style as white,
# dark, whitegrid, darkgrid
# or ticks
sns.set(style="white")

# draw lineplot
sns.lineplot(x="sepal_length", y="sepal_width", data=data)

# Removing the spines
sns.despine()

plt.show()
  • Escalado de los grficos: Se puede hacer mediante el mtodo set_context(). Anula los parmetros por defecto, afectando el tamao de las etiquetas, lneas y otros elementos del grfico, pero no al estilo general. El contexto base es "notebook", y los otros contextos son "paper", "talk", y "poster". font_scale establece el tamao de la fuente.
